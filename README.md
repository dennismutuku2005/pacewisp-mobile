# PaceWisp Enterprise

**PaceWisp** is a premium, high-performance management application designed for Wireless Internet Service Providers (WISPs). It provides a sleek, enterprise-grade interface for monitoring network health, managing customer sessions, and tracking financial performance in real-time.

---

## 🚀 Key Features

### 📊 Intelligent Dashboard
- **Real-time Metrics**: Track daily earnings, monthly revenue, and active entries at a glance.
- **Dynamic Growth Charts**: Visualize network utilization and revenue trends using interactive line and bar charts.
- **Station Node Monitoring**: Monitor the live status and uptime of your network infrastructure.

### 🎫 Voucher Management
- **Quick Generation**: Create vouchers for various bandwidth tiers with a single tap.
- **Automated Workflows**: Streamline the process of plan assignment and voucher distribution.

### 💰 Financial Tracking
- **Revenue Analytics**: Detailed breakdowns of income across different routers and timeframes.
- **Transaction Logs**: Real-time activity feed showing live connections and payment confirmations.

### 🏠 Home Screen Widget
- **Glanceable Data**: View today's earnings and active users directly from your phone's home screen.
- **Privacy First**: Integrated "Blur Mode" allows you to hide sensitive financial data until you click the widget to reveal it.

### 🔒 Enterprise Security
- **Biometric App Lock**: Secure your data using system-level fingerprint or facial recognition.
- **Multi-Instance Support**: Easily switch between different service subdomains and accounts.
- **Adaptive UI**: High-contrast Dark Mode for reduced eye strain and premium aesthetics.

---

## 🛠 Technology Stack

- **Framework**: [Flutter](https://flutter.dev) (Cross-platform)
- **State Management**: [Provider](https://pub.dev/packages/provider)
- **Networking**: High-performance REST API integration
- **Storage**: Secure SharedPreferences with JSON serialization
- **Icons & Fonts**: Google Fonts (Figtree, JetBrains Mono) & Material Symbols

---

## 🎨 Design Philosophy

PaceWisp follows a **Premium Dark/Glassmorphism** aesthetic, utilizing deep purples and emerald accents to create an interface that feels alive, responsive, and professional. Every interaction is designed with micro-animations and subtle haptic feedback to ensure a state-of-the-art user experience.
